# Mac Setup
### Install Homebrew
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
### Install Warp
```
brew install --cask warp
```
### Install Raycast
```
brew install --cask raycast
```
